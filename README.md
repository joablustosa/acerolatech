# Acerola Tech — Consultoria Digital

Site institucional da Acerola Tech: consultoria digital B2B com foco em projetos fechados.

## Serviços

- Desenvolvimento de soluções (web e desktop)
- Apps mobile (iOS e Android)
- Agentes de automação com IA
- Integração de sistemas
- Modernização de sistemas legados

## Tecnologias

- React 18
- Vite
- Tailwind CSS v4
- React Router

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```
